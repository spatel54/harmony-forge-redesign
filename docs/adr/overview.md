# Architecture Decision Records — Index

Accepted ADRs are **immutable**. To reverse a decision, write a new superseding ADR.

## Lifecycle States

| State | Meaning |

| `Accepted` | In effect — immutable |
| `Superseded` | Replaced by a newer ADR (link provided) |
